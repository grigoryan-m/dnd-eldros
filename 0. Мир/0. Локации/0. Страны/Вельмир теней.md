**Население:** ~2 тыс.
**Основная раса:** Люди
**Безопасность:** 0

**Патруль:** -

Когда [[Манамоа и братья|братья]] освободились из [[Колодец Отчаяния|Колодца]], часть их [[Вельмир|вельмира]] просочилась в этот лес. Теперь лес является живой сущностью, намеренно мешая существам спокойно существовать в этом лесу.

Существа, живущие в резонансе с [[Вельмир|вельмиром]] братьев становятся могущественнее и сильнее.

Из [[Дерево Вельмира Теней|деревьев леса]] особо злые [[Корлюксы]] создают себе [[Посох|посохи]] (чаще всего - чернокнижники).

При создании [[Портал|портала]] из одной точки в другую через Вельмир Теней, есть шанс 30%, что второй портал окажется в самом лесу.
